# 使用方法

```kotlin
plugins {
    id("cn.rtast.rkmbed") version "<version>"
}

repositories {
    mavenCentral()
    maven("https://repo.maven.rtast.cn/releases")
}
```

> 最新版本可以在这里找到 https://maven.rtast.cn/#/releases/cn/rtast/rkmbed/gradle-plugin
